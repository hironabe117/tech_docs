## 認証周りのコマンド
```bash
# 設定
aws configure
# 確認
aws configure list
```
## profileの定義
`~/.aws/config`にてプロファイルを定義する。
```bash
[default]
region = ap-northeast-1

[profile EC2Role]
credential_source = Ec2InstanceMetadata
region = ap-northeast-1
```

`~/.aws/credentials`にて使用するアクセスキーとシークレットキーを指定する。
```bash
[default]
aws_access_key_id = AKIAXXXX
aws_secret_access_key =　XXX
```

## profileの切り替え
プロファイルを環境変数に設定し、CLI実行時に`--profile`で指定する。
```bash
export AWS_PROFILE=example_profile
aws s3 ls --profile example_profile
```

## 確認コマンド
下記のようなコマンドを用いて、現在のプロファイルの権限で何が実行できるか確認する。
```bash
aws ec2 describe-instances
aws s3 ls
```
